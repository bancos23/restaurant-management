\doxysubsection{Detailed Description}
Menu Page. 

This script fetches recipes and their ingredients from the database and displays them in a card layout.

The following steps are performed\+:
\begin{DoxyEnumerate}
\item Prepare and execute a SQL query to fetch recipe names, supply names, quantities, and unit names.
\item Fetch all results as objects and group them by recipe name.
\item Iterate over the grouped recipes and display each recipe and its ingredients in a card format.
\end{DoxyEnumerate}

Database Tables\+:
\begin{DoxyItemize}
\item recipies (r)\+: Contains recipe information.
\item recipies\+\_\+content (rc)\+: Contains the relationship between recipes and supplies, including quantities and units.
\item supplies (s)\+: Contains supply information.
\item units (u)\+: Contains unit information.
\end{DoxyItemize}

HTML Structure\+:
\begin{DoxyItemize}
\item A container with a row of cards, each representing a recipe.
\item Each card displays the recipe name and a list of ingredients with their quantities and units.
\item A pagination section at the bottom (currently static with only one page).
\end{DoxyItemize}

Dependencies\+:
\begin{DoxyItemize}
\item Config\+::get\+Con()\+: A method to get the database connection.
\item PDO\+::\+FETCH\+\_\+\+OBJ\+: Fetch mode to get results as objects.
\item Font\+Awesome icons for displaying utensils icon. 
\end{DoxyItemize}
