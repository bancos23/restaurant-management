\doxysubsection{Detailed Description}
Profile Page. 

This script fetches and displays the profile information of a user based on the username provided in the URL.

It performs the following steps\+:
\begin{DoxyEnumerate}
\item Prepares and executes a SQL query to fetch user details from the {\ttfamily users} table where the username matches the provided parameter.
\item If no user is found, it redirects to a 404 page.
\item If a user is found, it displays the user\textquotesingle{}s profile information including their full name, group, phone number, and email.
\end{DoxyEnumerate}

The profile information is displayed in a card format with the following details\+:
\begin{DoxyItemize}
\item Full name
\item Group
\item Phone number
\item Email
\end{DoxyItemize}

If the user does not exist, a 404 error page is displayed.
